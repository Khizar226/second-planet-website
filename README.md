# second-planet-website
Official website for Second Planet ($PLANET) — a Solana meme coin inspired by humanity’s need for a backup planet. Built as a single-file HTML/CSS/JS site and deployed via GitHub Pages.
Second Planet is a meme project inspired by Mars, civilization survival, and space culture. No utility. Just vision.
